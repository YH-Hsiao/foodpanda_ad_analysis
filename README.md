# Foodpanda 廣告效益分析專案

## 專案簡介  
本專案分析 Foodpanda 不同廣告渠道的投資回報率（ROAS），並結合營收與行銷花費資料，幫助優化行銷策略。

## 專案架構  
- data/foodpanda_sales.csv：模擬銷售與花費數據  
- analysis.py：Python 分析程式  
- sql_queries.sql：SQL 查詢腳本  
- report.md：完整分析報告  
- dashboard/：分析視覺化圖片  

## 技術堆疊  
Python (pandas, seaborn, matplotlib), Git, GitHub
